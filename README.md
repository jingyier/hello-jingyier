# jingyier

An artful, lightweight static site built with Astro and prepared for Cloudflare Pages plus a small Vercel JSON API.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Deployment

- Cloudflare Pages: build command `npm run build`, output directory `dist`.
- Vercel API: deploy the `api/` directory if runtime JSON endpoints are needed.
